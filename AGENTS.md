# AGENTS.md

## 프로젝트 목적
이 저장소는 김보민의 학습, 프로젝트, 대회, 데이터 구축 경험을 모아 **Data-Centric Robotics Engineer** 포트폴리오로 발전시키기 위한 작업 공간이다.

최종 목표는 단순한 활동 기록이 아니라, 다음 역량을 증거 기반으로 보여주는 것이다.

1. 로봇이 왜 현장에서 실패하는지 이해하는 사람
2. 데이터를 어떻게 모아야 하는지 아는 사람
3. 시뮬레이션과 실제 환경의 차이를 줄일 수 있는 사람
4. VLA 모델의 출력을 실제 로봇 제어로 연결할 수 있는 사람
5. 센서, 하드웨어, 제어, AI를 함께 이해하는 사람

## 포지셔닝
기본 포지션은 **Data-Centric Robotics Engineer**이다.

핵심 메시지:

> 현장 데이터를 이해하고, 센서·제어·AI를 연결해 로봇이 실제 환경에서 작동하도록 만드는 엔지니어를 목표로 한다.

## 포트폴리오 핵심 축
프로젝트와 자료는 다음 축으로 분류한다.

- Field Data: 현장 데이터 수집, AI데이터 구축, 라벨링, 품질관리
- Robotics AI: 센서, 인식, 제어, VLA-to-Control
- Sim2Real: 시뮬레이션과 실제 환경의 차이 분석
- Venture/Product: 벤처창업 경진대회, 문제정의, MVP, 사용자 검증

## 자료 관리 원칙
자료 정리의 기본 단위는 **프로젝트 카드**이다.

각 프로젝트 카드는 다음 구조를 따른다.

- 문제: 어떤 현장 문제를 다루는가
- 데이터: 어떤 데이터를 수집·정리·분석했는가
- 실험: 어떤 기술적 시도나 검증을 했는가
- 결과: 무엇을 배웠고 무엇이 개선됐는가
- 증거: 사진, 영상, 코드, 데이터 샘플, 문서, 발표자료, 회고

## 주요 활동 기록 방식

### 벤처창업 경진대회
벤처창업 경진대회는 Venture/Product 축의 대표 활동으로 관리한다.

기록할 내용:

- 문제정의
- 대상 사용자 또는 현장
- 제안한 솔루션
- 기술 구성
- 데이터 또는 AI 활용 방식
- MVP 또는 프로토타입
- 발표자료와 결과
- 배운 점과 다음 개선 방향

이 활동은 "기술을 실제 문제와 연결할 수 있는 사람"이라는 증거로 사용한다.

### 국가 AI데이터 구축 프로젝트
국가 AI데이터 구축 프로젝트는 Field Data 축의 핵심 활동으로 관리한다.

기록할 내용:

- 구축한 데이터의 목적
- 데이터 수집 방식
- 라벨링 기준
- 품질관리 방식
- 오류 사례와 개선 방법
- 데이터가 AI 모델 또는 로봇 시스템에 어떻게 쓰일 수 있는지
- 참여 과정에서 배운 데이터 중심 사고

이 활동은 "데이터를 어떻게 모아야 AI/로봇 시스템에 쓸 수 있는지 아는 사람"이라는 증거로 사용한다.

## 웹 포트폴리오 구성
웹 포트폴리오는 다음 섹션을 우선 구성한다.

- Home: Data-Centric Robotics Engineer 정체성
- Projects: 대표 프로젝트 카드
- Field Data: 데이터 수집·구축·품질관리 경험
- Robotics Stack: 센서, 제어, AI, 시뮬레이션 연결 역량
- Venture: 벤처창업 경진대회와 제품화 시도
- Growth Log: 1학년부터 2학년까지의 성장 기록

## Codex 작업 원칙
Codex는 이 프로젝트에서 다음 원칙을 따른다.

- 포트폴리오를 단순 이력서가 아니라 "증거 기반 성장 기록"으로 만든다.
- 새 자료를 추가할 때는 반드시 어떤 핵심 축에 속하는지 분류한다.
- 프로젝트 설명은 기술명 나열보다 문제, 데이터, 실험, 결과를 우선한다.
- 프로젝트 내용을 추가하거나 확정하기 전에는 반드시 김보민에게 실제 경험을 확인한다.
- 특히 어려웠던 점, 해결 과정, 강조하고 싶은 로직, 실패 사례, 직접 기여한 부분은 임의로 작성하지 않는다.
- 김보민이 확인하지 않은 내용은 초안 또는 가정으로 표시한다.
- 웹에 공개할 내용과 Drive/Notion에 보관할 원본 자료를 구분한다.
- 채용 담당자가 1분 안에 김보민의 방향성을 이해할 수 있도록 문장을 다듬는다.
- 벤처창업 경진대회와 국가 AI데이터 구축 프로젝트는 핵심 대표 활동으로 우선 반영한다.
